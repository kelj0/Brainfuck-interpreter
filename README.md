#### About
Interprets given brainfuck code

#### Usage
- Get bf code  ([here is a good page for that](https://copy.sh/brainfuck/text.html))
- Add bf code to bf.txt (there is already demo `Hello world!` in bf.txt)
- `$ g++ main.cpp`
- `$ ./a.out`
